## Project Overview
This project analyzes the Titanic passenger dataset to understand the factors that influenced survival.

## Objectives
- Analyze survival rate by gender, class, and age
- Perform data cleaning and preprocessing
- Explore patterns using pandas and numpy

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

## Dataset
- Titanic dataset (Kaggle)

## Notebook
- `titanic_survivor_analysis.ipynb`

## Key Insights
- Women had a significantly higher survival rate
- Passengers in higher classes were more likely to survive
- Age and family size impacted survival probability
